# Objection Atlas

Objection Atlas is a calm one-screen utility for mapping the doubts standing between a promising idea and a believable yes.

It helps turn a rough pitch into:

- safer wording
- three concrete objections
- the missing proof to collect
- a tomorrow test
- a short talk track

## Local development

```bash
npm install
npm run dev
```

## Production check

```bash
npm run build
npm run start -- --hostname 127.0.0.1 --port 3132
```

Basic auth is controlled with:

- `BASIC_AUTH_USERNAME`
- `BASIC_AUTH_PASSWORD`
